# port_scanner_and_threat_monitoring_tool
Cyber security tool for scanning port and monitoring network threats


## How to clone the repo
```bash
git clone https://github.com/Iankulani/port_scanner_and_threat_monitoring_tool.git
cd port_scanner_and_threat_monitoring_tool
```

## How to run
```bash
python port_scanner_and_threat_monitoring_tool.py
```
